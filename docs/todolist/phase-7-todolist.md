Dokumen ini merinci pengembangan laporan lanjutan dan analisis data untuk memberikan wawasan (insight) yang lebih mendalam bagi para pemimpin organisasi.

## 1. Matriks Kehadiran (Attendance Grid)
*   [ ] **Visualisasi Grid:** Tabel dengan baris Nama Anggota dan kolom Tanggal Pertemuan.
*   [ ] **Filter Periode:** Membatasi kolom tanggal berdasarkan range waktu tertentu.
*   [ ] **Quick Look:** Ringkasan kehadiran di akhir baris anggota.

## 2. Analisis Performa Grup (Group Insights)
*   [ ] **Dashboard Leaderboard:** Ranking kelompok berdasarkan persentase kehadiran.
*   [ ] **Aggregation Report:** Rekapitulasi per tingkatan grup (Nasional vs Wilayah vs Kelompok).

## 3. Sistem "Early Warning" (Anggota Kurang Aktif)
*   [ ] **Laporan Low Participation:** Anggota dengan kehadiran di bawah threshold (misal: < 50%).
*   [ ] **Detection Trend:** Mendeteksi anggota yang sudah absen 3 kali berturut-turut.

## 4. Pelaporan Lanjutan (Advanced Reporting)
*   [ ] **Export Laporan Bulanan PDF:** Layout rapi untuk rapat bulanan.
*   [ ] **Breakdown Status Detail:** Statistik Sakit vs Izin vs Alpa.
*   [ ] **Penyajian Data Berbasis Kategori Usia:** Analisis tingkat keaktifan per kategori.

## 5. Definition of Done (DoD) - Phase 7
1.  Admin dapat melihat pola kehadiran anggota dalam bentuk matriks tanggal.
2.  Tersedia daftar "Anggota Butuh Perhatian" berdasarkan data absensi berturut-turut.
3.  Pemimpin grup dapat membandingkan performa kehadiran antar kelompok di bawahnya.
4.  Laporan bulanan siap cetak dalam format PDF dengan desain profesional.
